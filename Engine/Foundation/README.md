# Foundation
> + 정적 라이브러리
> + MultiPlatform을 위한 LowLevel 추상화 함수 구현

## 규칙
> + 미리 컴파일된 헤더는 pch.h 이름을 가진다
> + 모든 구현사항은 fd namespace에 종속
> + 모든 매크로 함수는 FD_* prefix 사용
> + fd namespace는 트리 레벨0
> + 상위레벨은 하위레벨에 접근불가. 하위레벨은 상위레벨에 접근가능. 형제레벨 접근가능

## CMake
> + 소스코드가 있는 모든 폴더에는 포함관리를 위해 CMakeLists 파일이 존재하나 Foundation 폴더를 제외한
    CMakeLists 파일에는 최종출력에 관한 내용이 없어야 한다
> + 위와 같은 포함 시스템을 사용하기 위해 강제적으로 CMake 3.16 버전 사용이 강제됨.
    (하위 버전에서도 구현이 가능하나 깔끔함을 위해 3.16을 강제)
> + 반드시 GLOB을 사용하지 않고 수기로 파일리스트를 작성할 것
> + 미리 컴파일된 헤더는 CMake의 기능을 이용하므로 CPP파일에는 포함하지 말 것. CMake가 자동으로 포함 처리
